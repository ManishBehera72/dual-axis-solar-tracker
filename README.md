# Dual Axis Solar Tracker

## Overview
This project is an Arduino-based Dual Axis Solar Tracker designed to maximize solar energy absorption by continuously aligning a solar panel toward the direction of maximum sunlight. The system uses LDR sensors to detect light intensity and servo motors to move the panel along both horizontal and vertical axes.

The mechanical structure of the tracker is designed using 3D printed parts for lightweight construction and smooth movement.

---

## Features
- Automatic sunlight tracking
- Dual-axis movement
- Arduino Uno control
- LDR-based light sensing
- Servo motor actuation
- 3D printed mechanical design
- Energy-efficient operation

---

## Components Used
- Arduino Uno
- 4 LDR Sensors
- 2 Servo Motors
- Solar Panel
- Breadboard
- Jumper Wires
- External Power Supply
- 3D Printed Parts

---

## Working Principle
The LDR sensors detect the intensity of sunlight from different directions. The Arduino compares the sensor values and controls the servo motors accordingly to rotate the solar panel toward the brightest light source.

This improves solar panel efficiency compared to fixed solar panels.
